## Types of Machine Learning

##

There are **4 main types** of Machine Learning:

1.  Supervised Learning
2.  Unsupervised Learning
3.  Semi-Supervised Learning
4.  Reinforcement Learning

---

## 1\. Supervised Learning

### What is it?

##

Learning using **labeled data**.

ðŸ‘‰ Data has **input + correct output**.

### Example

##

| Input      | Output          |
| ---------- | --------------- |
| Email text | Spam / Not Spam |
| House size | Price           |

### Common Algorithms

##

- Linear Regression
- Logistic Regression
- Decision Tree
- Random Forest
- SVM

### Used for:

##

- Classification
- Regression

ðŸ“Œ **Interview line:**  
Supervised learning uses labeled data to train a model to predict correct outputs.

---

## 2\. Unsupervised Learning

### What is it?

##

Learning from **unlabeled data**.

ðŸ‘‰ No correct answer is given.

### Example

##

- Grouping customers by behavior
- Finding patterns in data

### Common Algorithms

##

- K-Means
- Hierarchical Clustering
- DBSCAN
- PCA

### Used for:

##

- Clustering
- Dimensionality reduction

ðŸ“Œ **Interview line:**  
Unsupervised learning finds hidden patterns in unlabeled data.

---

## 3\. Semi-Supervised Learning

### What is it?

##

Combination of **small labeled data + large unlabeled data**.

### Example

##

- 100 labeled images
- 10,000 unlabeled images

Used when labeling data is expensive.

### Applications

##

- Image recognition
- Speech recognition

ðŸ“Œ **Interview line:**  
Semi-supervised learning uses both labeled and unlabeled data for training.

---

## 4\. Reinforcement Learning

### What is it?

##

Learning by **trial and error** using **rewards and penalties**.

### Key Components

##

- Agent (Learner)
- Environment
- Action
- Reward

### Example

##

- Chess
- Games
- Robot navigation

ðŸ“Œ **Interview line:**  
Reinforcement learning trains an agent to take actions that maximize rewards.
