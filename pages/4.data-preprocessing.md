## 2\. Handling Missing Values

# 

Missing values = empty data (NaN).

### Ways to handle missing data:

#### 1\. Remove rows

# 

df.dropna()

Use when missing values are very few.

#### 2\. Fill with mean / median / mode

# 

df.fillna(df.mean())

#### 3\. Forward / Backward fill

# 

df.fillna(method\="ffill")

ðŸ“Œ **Interview tip:**  
Median is better when data has outliers.

## 5\. Feature Scaling

# 

Feature scaling brings all values to the **same range**.

### Why needed?

# 

*   Some features have big values
    
*   Some have small values
    
*   Model gets confused
    

ðŸ“Œ Required for:

*   KNN
    
*   SVM
    
*   Gradient Descent
    

* * *

## 6\. Normalization

# 

Scales values between **0 and 1**.

Formula:

(x - min) / (max - min)

### Example

# 

*   Age: 0 to 1
    
*   Height: 0 to 1
    

ðŸ“Œ Used when data has no outliers.

* * *

## 7\. Standardization

# 

Transforms data to:

*   Mean = 0
    
*   Standard deviation = 1
    

Formula:

(x - mean) / std

ðŸ“Œ Used when data has outliers and follows normal distribution.

* * *

## 8\. Train-Test Split

# 

Split data into:

*   Training data (learn)
    
*   Testing data (evaluate)
    

Common split:

80% Train

20% Test

Example:

from sklearn.model\_selection import train\_test\_split

X\_train, X\_test, y\_train, y\_test \= train\_test\_split(X, y, test\_size\=0.2)

ðŸ“Œ Never test on training data.

## 9\. Data Imbalance Problems

### What is it?

# 

One class has **more data** than others.

### Example

# 

*   Fraud: 1%
    
*   Normal: 99%
    

### Problems

# 

*   Model becomes biased
    
*   Poor prediction for minority class
    

### Solutions

# 

*   Oversampling (SMOTE)
    
*   Undersampling
    
*   Use proper evaluation metrics
    

ðŸ“Œ **Interview line:**  
Data imbalance can be handled using resampling techniques.
