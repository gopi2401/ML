*   Large
    
*   Noisy
    
*   Inconsistent
    

Preprocessing makes images **model-ready**.

## 3\. Object Detection

### What is Object Detection?

# 

*   Detects **what object**
    
*   Detects **where it is** (bounding box)
    

* * *

### Example

# 

*   Detect people in a photo
    
*   Detect cars on road
    

* * *

### Popular Algorithms

# 

*   YOLO
    
*   SSD
    
*   Faster R-CNN
    

ðŸ“Œ **Interview line:**

## 5\. CNNs for Images

### Why CNNs are used?

# 

CNNs are good at:

*   Capturing spatial features
    
*   Handling image data efficiently
    

* * *

### Key CNN Components (Quick)

# 

| Layer | Purpose |
| --- | --- |
| Convolution | Feature extraction |
| Pooling | Reduce size |
| Fully Connected | Prediction |

* * *

### Advantages of CNNs

# 

*   Fewer parameters
    
*   High accuracy
    
*   Automatic feature learning
    

ðŸ“Œ **Interview line:**  
CNNs are best suited for image-related tasks.
